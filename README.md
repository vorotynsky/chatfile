# Chatfile

**Chatfile** is a simple prompt-driven tool for interacting with language models (LLMs) using plaintext. 

## Example

```plaintext
FROM chatgpt
SYSTEM You are a technical writer.

ASK Write a readme file for the chatfile project
ANSWER |
    **Chatfile** is a simple prompt-driven tool for interacting with language models (LLMs) using plaintext. 
```
